# TalentTrack

TalentTrack is a mini dashboard application built using **Next.js App Router** and **Chakra UI v3** to demonstrate modern full-stack patterns.

## Features

- Authentication with server-side cookies
- Protected dashboard routes
- Skills management (CRUD)
- Tasks management with status workflow
- Server Actions (no API routes)
- Chakra UI v3 theming & components

## Tech Stack

- Next.js (App Router)
- React Server Components
- Chakra UI v3
- Server Actions
- Middleware

## What I Learned

- Server vs Client Components
- App Router architecture
- Server Actions and redirects
- Chakra UI v3 Dialogs and theming
- Building production-grade dashboards
